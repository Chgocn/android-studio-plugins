## 插件

### 整合

- [Chgocn-AndroidStudio-插件备份集](https://github.com/Chgocn/android-studio-plugins)

### 使用

1. 进入C:\Users\yourComputerUserName\.AndroidStudio2.X\config\plugins

        git clone https://github.com/Chgocn/android-studio-plugins

2. 重启Android Studio
